# Changelog

All notable changes to the Sunrise Painting Co. website will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2023-10-XX

### Added
- **Initial Release**
  - Responsive Homepage with split-screen hero
  - Services listing page with detailed pricing
  - About Us page with company history and values
  - Gallery page with masonry grid layout
  - Contact page with integrated Smart Estimate Form
  - FAQ page with accordion interactions
  - Interactive Before/After slider component for the gallery
  - Multi-step "Smart Estimate" form with validation
  - Sticky header navigation with mobile hamburger menu
  - Footer with business info, hours, and links
  - Trust badges and social proof sections
  - SEO Optimization (Sitemap, Robots.txt, Meta tags)
  - Structured Data (JSON-LD) for LocalBusiness
  - Centralized Image Configuration system (`src/config/images.ts`)
  - Accessibility features (ARIA labels, keyboard navigation)
  - Design System implementation (Colors, Typography, Spacing)
